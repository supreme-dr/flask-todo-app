# Flask To-Do App

A full-stack To-Do web application built with **Flask**, **HTML**, **CSS**, and **SQLite**.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Persistent storage using SQLite
- Clean and responsive UI

## Tech Stack
- Python (Flask)
- HTML & CSS
- SQLite
- SQLAlchemy

## How to Run Locally
```bash
pip install flask flask_sqlalchemy
python app.py
